= 疑難排解
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
升級節點配對時可能會發生故障。節點可能會當機、集合體可能不會重新部署、或是生命體可能無法移轉。故障原因及其解決方案取決於升級程序期間何時發生故障。

請參閱本節中說明程序不同階段的表格 link:overview_of_the_arl_upgrade.html["ARL升級總覽"]。程序階段會列出可能發生的故障相關資訊。
