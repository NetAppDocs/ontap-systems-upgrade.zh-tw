= 淘汰節點1
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
若要淘汰節點1、請繼續執行自動化作業、以停用與節點2的HA配對、並正確關閉節點1。您必須稍後從 node1 機箱中移除舊的系統模組、然後在 node1 上安裝替換模組。

.步驟
. 恢復作業：
+
「系統控制器更換恢復」

. 驗證node1是否已停止：
+
「System Controller replace show-details」（系統控制器取代顯示詳細資料

+
節點1完全停止後、node1應位於loader>提示字元。若要查看Loader>提示、請連線至node1的序列主控台。
