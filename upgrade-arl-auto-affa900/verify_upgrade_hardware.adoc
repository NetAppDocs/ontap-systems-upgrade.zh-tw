= 驗證升級硬體
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
在開始升級之前、您必須先確認AFF 您擁有的是正確的硬體、可讓您進行從故障到AFF 故障的修復A700升級、或是從FAS9000升級到FAS9500升級。對於要升級的每個HA配對、您必須有兩個系統控制器模組和兩個NVRAM模組。如果零件遺失、請聯絡技術支援部門或NetApp銷售聯絡人以尋求協助。
