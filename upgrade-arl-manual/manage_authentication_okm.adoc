= 使用 Onboard Key Manager 管理驗證金鑰
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您可以使用 Onboard Key Manager （ OKM ）來管理驗證金鑰。如果您已設定 OKM 、則必須在開始升級之前記錄複雜密碼和備份資料。

.步驟
. 記錄叢集範圍的複雜密碼。
+
這是使用 CLI 或 REST API 設定或更新 OKM 時所輸入的複雜密碼。

. 驗證通關密語和金鑰管理程式操作：
+
「安全金鑰管理程式內建同步」

+
出現提示時，輸入您在上一個步驟中記錄的叢集範圍 OKM 複雜密碼。

. 透過查看以下命令的輸出來驗證 cc-mode 是否已啟用：
+
`security key-manager config show`

. 備份金鑰管理器資訊：
+
「安全金鑰管理程式內建show Backup」
