= 階段5.安裝及開機節點4
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
在第5階段期間、您會安裝並開機node4、將叢集和節點管理連接埠從node2對應至node4、然後驗證node4安裝。您也可以將節點2擁有的資料生命週期和SAN生命週期從節點3移至節點4、然後將節點2的集合體從節點3重新定位至節點4。

. link:map_ports_node2_node4.html["將連接埠從節點2對應至節點4"]
. link:verify_node4_installation.html["驗證node4安裝"]
. link:move_nas_lifs_node2_from_node3_node4_verify_san_lifs_node4.html["將節點2擁有的NAS資料生命週期從節點3移至節點4、並驗證節點4上的SAN生命週期"]
