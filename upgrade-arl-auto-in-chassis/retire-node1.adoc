= 淘汰節點1
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
若要淘汰 node1 、您可以恢復自動作業、以停用 HA 與 node2 配對、並正確關閉 node1 。

.步驟
. 恢復作業：
+
「系統控制器更換恢復」

. 驗證node1是否已停止：
+
「System Controller replace show-details」（系統控制器取代顯示詳細資料

+
節點1完全停止後、node1應位於loader>提示字元。若要查看Loader>提示、請連線至node1的序列主控台。



.接下來呢？
在節點 1 退役後，根據您的升級組態，選擇下列其中一個控制器更換程序：

* link:replace-node1-affa250-affc250.html["更換節點 1 AFF A250 或 AFF C250 控制器模組"]
* link:replace-node1-affa800.html["更換節點 1 AFF A800 或 AFF C800 控制器模組"]
* link:replace-node1-affa220-affa200-affc190-fas2620-fas2720.html["更換節點 1 AFF A220、AFF A200、AFF C190、FAS2620 或 FAS2720 控制器模組"]
* link:replace-node1-aff700-fas900.html["更換節點 1 的 AFF A700 或 FAS9000 控制器和 NVRAM 模組"]
